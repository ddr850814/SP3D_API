## 1. 公共初始化代码块

所有空间管理示例的起点都是获取工厂、模型、目录和根空间对象。以下是公共初始化代码：

```vb
Dim oPlant As Plant = MiddleServiceProvider.SiteMgr.ActiveSite.ActivePlant
Dim oModel As Model = oPlant.PlantModel
Dim oCatalog As Catalog = oPlant.PlantCatalog
Dim oRootSpace As BusinessObject = DirectCast(oModel.RootSpace, BusinessObject)
Dim oCatHelper As New CatalogBaseHelper(oCatalog)
Dim oSOMPart As Part = DirectCast(oCatHelper.GetPart("SPACE_DEF_SA01"), Part)
```

> **关键说明**：`SPACE_DEF_SA01` 是区域空间（AreaSpace）的零件号。不同空间类型使用不同零件号（见第 6 节对比表）。

## 2. 七种 Space 构造方式（含代码）

Smart3D 空间对象通过 `SpaceConstructionType` 枚举指定构造方式，配合 `Collection(Of SpaceInputs)` 传入输入参数。

### 2.1 SpaceBy2Points — 两点构造法

通过两个对角点定义空间范围，支持 `AssociatedObject` 设备关联：

```vb
Dim oParentColl As New Collection(Of SpaceInputs)()
Dim oSpaceInputs As SpaceInputs() = New SpaceInputs(2) {}
oSpaceInputs(0) = New SpaceInputs()
oSpaceInputs(1) = New SpaceInputs()

' 创建持久化点
Dim P1 As New Point3d(oModel, 1, 0, 0)
Dim P2 As New Point3d(oModel, 1.5, 0.5, 0.5)

oSpaceInputs(0).InputObject = P1
oSpaceInputs(1).InputObject = P2

oParentColl.Add(oSpaceInputs(0))
oParentColl.Add(oSpaceInputs(1))

' 创建空间
Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)
oSArea.SetPropertyValue("2PointsVolume1", "IJNamedItem", "Name")

oTransactionMgr.Commit("Place byPoints Volume")
```

### 2.2 SpaceBy4Points — 四点构造法

通过四个点精确定义空间底面：

```vb
Dim oSpaceInputs As SpaceInputs() = New SpaceInputs(4) {}
oSpaceInputs(0) = New SpaceInputs()
oSpaceInputs(1) = New SpaceInputs()
oSpaceInputs(2) = New SpaceInputs()
oSpaceInputs(3) = New SpaceInputs()

' 四个底面点
Dim P41 As New Point3d(oModel, 0, 0, 0)
Dim P42 As New Point3d(oModel, 0.5, 0, 0)
Dim P43 As New Point3d(oModel, 0.5, 0.5, 0)
Dim P44 As New Point3d(oModel, 0.5, 0.5, 0.5)

oSpaceInputs(0).InputObject = P41
oSpaceInputs(1).InputObject = P42
oSpaceInputs(2).InputObject = P43
oSpaceInputs(3).InputObject = P44

' 添加到集合
oParentColl.Add(oSpaceInputs(0))
oParentColl.Add(oSpaceInputs(1))
oParentColl.Add(oSpaceInputs(2))
oParentColl.Add(oSpaceInputs(3))

Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceBy4Points, oRootSpace, oParentColl)
oSArea.SetPropertyValue("SOMATPby4PointsVolume", "IJNamedItem", "Name")
```

### 2.3 SpaceByPlaneOffset — 平面与偏移构造法

支持五种 `PlaneDefinition` 类型。第一种使用 `Plane3d` 对象，后四种使用 `PlaneDefinition` 类：

**方式一：直接使用 Plane3d 对象**

```vb
' 创建三点平面
Dim oPlaneColl As New Collection(Of Position)()
oPlaneColl.Add(New Position(4, 0, 0))
oPlaneColl.Add(New Position(4, 4, 0))
oPlaneColl.Add(New Position(4, 4, 4))
Dim oPlane As New Plane3d(oModel, oPlaneColl)

oSpaceInputs(0).InputObject = oPlane
oSpaceInputs(0).Offsets.Add(1)    ' 偏移值1
oSpaceInputs(0).Offsets.Add(0.5)  ' 偏移值2

oSpaceInputs(1).InputObject = New Point3d(oModel, 1, 0, 0)
oSpaceInputs(2).InputObject = New Point3d(oModel, 1, 2, 0)
oSpaceInputs(3).InputObject = New Point3d(oModel, 1, 1, 1)

Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceByPlaneOffset, oRootSpace, oParentColl)
```

**方式二：PlaneDefinition.ThreePointsPlane（三点平面）**

```vb
Dim point3dCollection As New Collection(Of IPoint)()
point3dCollection.Add(New Point3d(oModel, 4.0, 0.0, 0.0))
point3dCollection.Add(New Point3d(oModel, 4.0, 4.0, 0.0))
point3dCollection.Add(New Point3d(oModel, 4.0, 4.0, 4.0))

Dim planeDef As New PlaneDefinition(point3dCollection)
oSpaceInputs(0).PlaneDefinition = planeDef
oSpaceInputs(0).Offsets.Add(1.0)
oSpaceInputs(0).Offsets.Add(0.5)
```

**方式三：PlaneDefinition.Coincident（重合平面）**

基于已有 GridPlane 创建重合平面：

```vb
' 先创建 GridPlane 作为参考
Dim CoordSys As New CoordinateSystem(oModel, CoordinateSystem.CoordinateSystemType.Grids)
Dim yAxis As New GridAxis(CoordSys, AxisType.Y)
Dim planeY1 As New GridPlane(-2.0, yAxis)

planeDef = New PlaneDefinition(planeY1)
oSpaceInputs(0).PlaneDefinition = planeDef
```

**方式四：PlaneDefinition.Offset（偏移平面）**

基于参考平面偏移一定距离：

```vb
planeDef = New PlaneDefinition(planeY1, 4.0)  ' 参考平面 + 偏移距离
oSpaceInputs(0).PlaneDefinition = planeDef
```

**方式五：PlaneDefinition.Angle（角度平面）**

绕旋转轴旋转一定角度：

```vb
Dim dAngle As Double = Math.PI / 15
Dim rotationAxisStartPoint As IPoint = New Point3d(oModel, 1.0, -2.0, 0.0)
Dim rotationAxisEndPoint As IPoint = New Point3d(oModel, 4.0, -2.0, 0.0)

planeDef = New PlaneDefinition(planeY1, rotationAxisStartPoint, rotationAxisEndPoint, dAngle)
oSpaceInputs(0).PlaneDefinition = planeDef
```

**方式六：PlaneDefinition.NormalToPlaneAndVector（法向平面）**

由法向量和面上一点定义：

```vb
Dim normPt1 As IPoint = New Point3d(oModel, 0, 0, 0)
Dim normPt2 As IPoint = New Point3d(oModel, 0, 0, 1)
Dim pointOnPlane As IPoint = New Point3d(oModel, 0, 0, 10)

planeDef = New PlaneDefinition(normPt1, normPt2, pointOnPlane)
oSpaceInputs(0).PlaneDefinition = planeDef
```

### 2.4 SpaceByPrimitive — 基本体构造法

使用目录中的预定义几何形状（如圆柱体）：

```vb
' 获取形状零件
Dim oSOMShape As Part = DirectCast(oCatalogBaseHelper.GetPart("RtCircularCylinder 001"), Part)

' 放置位置
Dim oPoint As New Point3d(4, 4, 4)

oSpaceInputs(0).InputObject = oSOMShape
oSpaceInputs(1).InputObject = oPoint

Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceByPrimitive, oRootSpace, oParentColl)
oSArea.SetUserDefinedName("SOMATPbyPrimitiveVolume")
```

### 2.5 MergeBySpace — 合并构造法

将多个已有空间通过布尔运算合并：

```vb
' 先创建多个基础空间（使用两点构造）
Dim oVolumeColl As New Collection(Of BusinessObject)()
For i = 1 To 2
    ' 创建两点构造空间并添加到集合
    Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)
    oVolumeColl.Add(oSArea)
    oTransactionMgr.Commit("Place byPoints Volume")
    oParentColl.Clear()
Next

' 设置布尔运算类型并合并
For j = 0 To 1
    oSpaceInputs = New SpaceInputs()
    oSpaceInputs.InputObject = oVolumeColl(j)
    oSpaceInputs.BooleanOperationType = BooleanOperationType.UnionOperation
    oParentColl.Add(oSpaceInputs)
Next

Dim oSMergeArea As New AreaSpace(oSOMPart, SpaceConstructionType.MergeBySpace, oRootSpace, oParentColl)
oSMergeArea.SetUserDefinedName("MergeVolume")
```

### 2.6 BoundBySpace — 边界构造法

由多个空间的边界约束定义新空间，结构与合并法类似：

```vb
For j = 0 To 1
    oSpaceInputs = New SpaceInputs()
    oSpaceInputs.InputObject = oVolumeColl(j)
    oSpaceInputs.BooleanOperationType = BooleanOperationType.UnionOperation
    oParentColl.Add(oSpaceInputs)
Next

Dim oSBoundArea As New AreaSpace(oSOMPart, SpaceConstructionType.BoundBySpace, oRootSpace, oParentColl)
oSBoundArea.SetPropertyValue("BoundVolume", "IJNamedItem", "Name")
```

### 2.7 SpaceByPath — 3D路径构造法

沿 Sketch3D 路径扫掠截面创建空间，支持两种截面类型：

**方式一：目录截面（Catalog Cross-Section）**

```vb
' 创建 3D 路径
Dim oSketchPt1 As New SketchPoint(7.0, 4.0, 0.0)
Dim oSketchPt2 As New SketchPoint(7.0, 4.0, 5.0)
Dim oSketchPt3 As New SketchPoint(7.0, 0.0, 5.0)
Dim oSketchPt4 As New SketchPoint(10.0, 0.0, 5.0)

oSketchPt1.CreateLineFeature(oSketchPt2)
oSketchPt2.CreateLineFeature(oSketchPt3)
oSketchPt3.CreateLineFeature(oSketchPt4)

Dim oSketch3DPath As New Sketch3D(oModel)
Dim oSketchPointColl As New Collection(Of SketchPoint)()
oSketchPointColl.Add(oSketchPt1)
oSketchPointColl.Add(oSketchPt2)
oSketchPointColl.Add(oSketchPt3)
oSketchPointColl.Add(oSketchPt4)
oSketch3DPath.SetSketchPoints(oSketchPointColl)

' 从目录获取截面
Dim oCatalogXSection As New SketchCrossSection(oModel)
Dim oXSection As Part = oCatalogBaseHelper.GetPart("SKETCH_CROSS_SECTION_01")
oCatalogXSection.CrossSectionDefinition = oXSection

oSpaceInputs(0).InputObject = oSketch3DPath
oSpaceInputs(1).InputObject = oCatalogXSection

Dim oSArea1 As New AreaSpace(oSPart, SpaceConstructionType.SpaceByPath, oParantSpace, oParentColl)
oSArea1.SetUserDefinedName("VolumeByPath_CatalogXSection")
```

**方式二：草图截面（Sketch Cross-Section）**

用 Sketch3D 手动绘制截面形状：

```vb
' 绘制截面轮廓（8个点形成 U 形截面）
Dim oXSectPt1 As New SketchPoint(12.0, 5.0, 0.0)
Dim oXSectPt8 As New SketchPoint(12.0, 3.0, 0.0)
' ... 其他点 ...
oXSectPt1.CreateLineFeature(oXSectPt2)
' ... 其他线段 ...
oXSectPt8.CreateLineFeature(oXSectPt1)  ' 闭合

Dim oSketch3DXSection As New Sketch3D(oModel)
oSketch3DXSection.SetSketchPoints(oSketchXSectionPointColl)

Dim oSketchXSection As New SketchCrossSection(oModel)
oSketchXSection.CrossSectionDefinition = oSketch3DXSection

oSpaceInputs(0).InputObject = oSketch3DPath
oSpaceInputs(1).InputObject = oSketchXSection

Dim oSArea2 As New AreaSpace(oSPart, SpaceConstructionType.SpaceByPath, oParantSpace, oParentColl)
oSArea2.SetUserDefinedName("VolumeByPath_SketchXSection")
```

## 3. SpaceInputs 使用模式

`SpaceInputs` 是空间构造的核心数据结构，包含以下关键属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| `InputObject` | `Object` | 输入对象（Point3d、Plane3d、Part、Sketch3D、BusinessObject 等） |
| `PlaneDefinition` | `PlaneDefinition` | 平面定义（仅 SpaceByPlaneOffset 使用） |
| `Offsets` | `Collection(Of Double)` | 偏移值列表（仅 SpaceByPlaneOffset 使用） |
| `BooleanOperationType` | `BooleanOperationType` | 布尔运算类型（仅 Merge/Bound 使用） |

### 使用模式总结

```
SpaceInputs 使用模式:
│
├── SpaceBy2Points    → 2个 SpaceInputs，InputObject = Point3d
├── SpaceBy4Points    → 4个 SpaceInputs，InputObject = Point3d
├── SpaceByPlaneOffset→ 1个 SpaceInputs(Plane+Offsets) + 3个 SpaceInputs(Point3d)
│     ├── InputObject = Plane3d  或  PlaneDefinition
│     ├── Offsets.Add(偏移值1)
│     └── Offsets.Add(偏移值2)
├── SpaceByPrimitive  → 2个 SpaceInputs：InputObject = Part(Shape) + Point3d
├── MergeBySpace      → N个 SpaceInputs：InputObject = BusinessObject(Space) + BooleanOperationType
├── BoundBySpace      → N个 SpaceInputs：InputObject = BusinessObject(Space) + BooleanOperationType
└── SpaceByPath       → 2个 SpaceInputs：InputObject = Sketch3D(Path) + SketchCrossSection
```

## 4. GetSpaceInputs / SetSpaceInputs 模式

创建空间后，可以通过 `GetSpaceInputs` 读回输入参数，修改后用 `SetSpaceInputs` 重新设置：

```vb
' 读取当前输入参数
oParentColl = oSArea.GetSpaceInputs()
If oParentColl.Count = 0 Then
    ' 报错：无法获取输入
End If

' 修改输入（例如替换某个点）
oParentColl.Clear()
Dim oPointNew As New Point3d(oModel, 4, 0, 0.5)
Dim oPointNew2 As New Point3d(oModel, 4.5, 0.5, 1.5)

oSpaceInputs(0).InputObject = oPointNew
oSpaceInputs(1).InputObject = oPointNew2
oParentColl.Add(oSpaceInputs(0))
oParentColl.Add(oSpaceInputs(1))

' 重新设置输入
oSArea.SetSpaceInputs(oParentColl)
oTransactionMgr.Commit("Modify Volume")
```

### Part 属性

修改空间关联的零件（新旧零件的 PartClass 必须相同）：

```vb
Dim oOldPart As Part = oSArea.Part
Dim oSOMPart As Part = DirectCast(oCatalogBaseHelper.GetPart("SPACE_DEF_SA02"), Part)

' 检查零件类是否相同
If oOldPart.PartClass Is oSOMPart.PartClass Then
    oSArea.Part = oSOMPart
    oTransactionMgr.Commit("Modify Volume")
End If
```

### ConstructionType 属性

```vb
Dim EConstructionType As SpaceConstructionType = oSArea.ConstructionType
If EConstructionType = SpaceConstructionType.SpaceBy2Points Then
    ' 验证构造类型正确
End If
```

### AssociatedObject 属性

关联业务对象（如设备）到空间：

```vb
' 关联设备
oSArea.AssociatedObject = oEqp
oTransactionMgr.Commit("Association")

' 获取关联对象
Dim oEquip As BusinessObject = oSArea.AssociatedObject

' 移除关联
oSArea.AssociatedObject = Nothing
oTransactionMgr.Commit("Remove Association")
```

## 5. FolderSpace 层级管理

`FolderSpace` 用于组织管理空间对象的层级容器。通过 `ISpaceParent` 接口实现父子关系。

### 创建层级结构

```vb
' 获取根空间作为父节点
Dim oRootSpace As ISpaceParent = oModel.RootSpace

' 创建一级文件夹
Dim oFolder As New FolderSpace(oRootSpace)
oFolder.SetUserDefinedName("F1")
oTransactionMgr.Commit("Create Space Folder")

' 创建子文件夹 — 将 FolderSpace 转换为 ISpaceParent
Dim oChildParent As ISpaceParent = DirectCast(oFolder, ISpaceParent)
Dim oSubFolder As New FolderSpace(oChildParent)
oSubFolder.SetUserDefinedName("F11")
oTransactionMgr.Commit("Create Sub Folder")
```

### 层级结构示例

```
Root
├── F1
│   ├── F11
│   └── F12
├── F2
│   └── F21
│       └── F211
└── F3
    ├── F31
    └── F32
```

### SpaceParent / SpaceChildren / AddSpaceChild

```vb
' 获取父节点
Dim oSpaceParent As ISpaceParent = oSpaceFolder.SpaceParent

' 获取所有子节点
Dim oChildren As ReadOnlyCollection(Of ISpaceChild) = oSpaceFolder.SpaceChildren

' 添加子空间
oSpaceFolder.AddSpaceChild(oAnotherFolder)

' 修改父节点（移动文件夹）
oSpaceFolder.SpaceParent = oModel.RootSpace
```

### GetAllSpaces — 递归获取所有空间

获取文件夹下所有空间对象（包括子文件夹中的）：

```vb
Dim oSpaceColl As Collection(Of SpaceBase) = oSpaceFolder.GetAllSpaces()
```

### 在文件夹下创建空间

```vb
' 将文件夹作为父节点
oRootSpace = DirectCast(oColl(0), ISpaceParent)

' 创建空间并指定父文件夹
Dim oSArea As New AreaSpace(oSOMPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)
oSArea.SetPropertyValue("2PointsVolume1", "IJNamedItem", "Name")
oTransactionMgr.Commit("Place byPoints Volume")
```

## 6. 四种空间类型对比

Smart3D 提供四种空间类型，它们的 API 结构完全一致，仅使用不同的零件号：

| 空间类型 | 类名 | 零件号示例 | 说明 | 示例文件 |
|---------|------|-----------|------|---------|
| 区域空间 | `AreaSpace` | `SPACE_DEF_SA01` | 表示区域性的空间对象，最常用 | `SpaceArea.vb` |
| 空间文件夹 | `FolderSpace` | — | 层级容器，不表示物理空间 | `SpaceFolder.vb` |
| 干涉空间 | `InterferenceSpace` | `SPACE_DEF_IV01` | 表示干涉/碰撞区域 | `SpaceInterference.vb` |
| 分区空间 | `ZoneSpace` | `SPACE_DEF_HZ01` | 表示分区概念的空间对象 | `SpaceZone.vb` |

### 构造方式通用性

`AreaSpace`、`InterferenceSpace`、`ZoneSpace` 三者均支持相同的七种构造方式：

```vb
' 区域空间
Dim oArea As New AreaSpace(oSPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)

' 干涉空间（仅需更换类名和零件号）
Dim oInterference As New InterferenceSpace(oSPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)

' 分区空间（仅需更换类名和零件号）
Dim oZone As New ZoneSpace(oSPart, SpaceConstructionType.SpaceBy2Points, oRootSpace, oParentColl)
```

### 构造方式选择指南

```
Space 构造方式选择:
│
├── 简单矩形空间 → 两点构造 (SpaceBy2Points)
├── 精确底面空间 → 四点构造 (SpaceBy4Points)
├── 基于平面定义 → 平面偏移构造 (SpaceByPlaneOffset)
│   ├── ThreePointsPlane (三点平面)
│   ├── Coincident (重合)
│   ├── Offset (偏移)
│   ├── Angle (角度)
│   └── NormalToPlaneAndVector (法向)
├── 标准几何形状 → 基本体构造 (SpaceByPrimitive)
├── 组合空间     → 合并构造 (MergeBySpace)
├── 边界约束     → 边界构造 (BoundBySpace)
└── 扫掠形状     → 3D路径构造 (SpaceByPath)
    ├── 目录截面 (Catalog Cross-Section)
    └── 草图截面 (Sketch Cross-Section)
```
