## 1. Pipeline 和 PipeRun 创建模板

管道布线的第一步是创建 `Pipeline`（管道线系统）和 `PipeRun`（管道段）。以下两个辅助函数封装了标准的创建流程，来自 `PipeRun.vb` 中的 `PipeRunExample` 类。

```vb
' 创建 Pipeline
Private Function NewPipeLineSystem(sPipeLineSys As String) As Pipeline
    Dim oRootSystem As BusinessObject = TryCast(
        MiddleServiceProvider.SiteMgr.ActiveSite.ActivePlant.PlantModel.RootSystem, BusinessObject)
    Dim oPipeline As New Pipeline(oRootSystem)
    oPipeline.SetPropertyValue(sPipeLineSys, "IJNamedItem", "Name")
    Return oPipeline
End Function

' 创建 PipeRun
Private Function NewPipeRun(oPipeLineSystem As Pipeline, dNPD As Double, 
    sNPDUnitType As String, sPipeSpec As String, sRunName As String,
    dMaxTemp As Double, dMaxPressure As Double) As PipeRun
    Dim oPipeRun As New PipeRun(oPipeLineSystem, dNPD, sNPDUnitType, sPipeSpec)
    oPipeRun.SetPropertyValue(sRunName, "IJNamedItem", "Name")
    oPipeRun.SetPropertyValue(dMaxTemp, "IJProcessDataInfo", "DesignMaxTemp")
    oPipeRun.SetPropertyValue(dMaxPressure, "IJProcessDataInfo", "DesignMaxPressure")
    Return oPipeRun
End Function
```

**典型调用方式：**

```vb
Dim oPipeLine As Pipeline = NewPipeLineSystem("MyPipelineSystem")

Dim oPipeRun As PipeRun = NewPipeRun(
    oPipeLine, 10, "in", "1C0031", "PipeRun1", 310, 4788)
```

> **关键参数说明**：`dNPD` = 公称直径数值，`sNPDUnitType` = 单位类型（如 `"in"`、`"mm"`），`sPipeSpec` = 管道规范编号，`dMaxTemp`/`dMaxPressure` = 设计最大温度/压力。

此外，`PipeRun` 还支持**复制构造函数**：

```vb
Dim oSecondPipeRun As New PipeRun(oFirstPipeRun)
```

## 2. 十种布线方法（RouteBetween***）

`PipeRun` 类提供十种 `RouteBetween***` 方法，覆盖所有可能的起点/终点组合（端口 Port、分支 Branch、末端特征 EndFeature、空间点 Point）。

| # | 方法签名 | 说明 |
|---|---------|------|
| 1 | `RouteBetweenPointAndPoint(fromPoint, toPoint, ByRef endFeature)` | **点到点** — 从一个空间点到另一个空间点布线，返回末端特征 |
| 2 | `RouteBetweenPortAndPort(port1, port2)` | **端口到端口** — 从设备端口到另一设备端口布线 |
| 3 | `RouteBetweenPortAndBranch(port, feature, position)` | **端口到分支** — 从设备端口到已有直管特征上的指定位置布线 |
| 4 | `RouteBetweenPortAndPoint(port, position, ByRef endFeature)` | **端口到点** — 从设备端口到空间点布线 |
| 5 | `RouteBetweenPortAndEndFeature(port, endFeature)` | **端口到末端特征** — 从设备端口到已有末端特征布线 |
| 6 | `RouteBetweenBranchAndBranch(feature1, pos1, feature2, pos2)` | **分支到分支** — 两个直管特征之间的布线（接管/支管） |
| 7 | `RouteBetweenBranchAndPoint(feature, fromPoint, toPoint, ByRef endFeature)` | **分支到点** — 从直管特征上某点到空间点布线 |
| 8 | `RouteBetweenBranchAndEndFeature(feature, fromPoint, endFeature)` | **分支到末端特征** — 从直管特征到已有末端特征布线 |
| 9 | `RouteBetweenEndFeatureAndPoint(endFeature, toPoint, ByRef newEndFeature)` | **末端特征到点** — 从末端特征延伸到空间点 |
| 10 | `RouteBetweenEndFeatureAndEndFeature(endFeature1, endFeature2)` | **末端特征到末端特征** — 连接两个末端特征 |

### 代码示例：点到点布线

```vb
Dim oFromPoint As New Position(121, 0, 0)
Dim oToPoint As New Position(129, 0, 0)
Dim oPipeEndFeature As PipeEndFeature = Nothing

oPipeRun.RouteBetweenPointAndPoint(oFromPoint, oToPoint, oPipeEndFeature)
MiddleServiceProvider.TransactionMgr.Commit("RouteLeg from 121,0,0 to 129,0,0.")
```

### 代码示例：端口到端口布线

```vb
' 从两台设备获取端口
Dim oDistribPort1, oDistribPort2 As DistributionPort = Nothing
GetPipePortFromEquipment(New Position(85, 3, 3), "Suction", oDistribPort1)
GetPipePortFromEquipment(New Position(89, 3, 3), "Suction", oDistribPort2)

oPipeRun.RouteBetweenPortAndPort(oDistribPort1, oDistribPort2)
MiddleServiceProvider.TransactionMgr.Commit("RouteBetweenPortAndPort Example.")
```

### 代码示例：端口到分支布线

先通过点到点布线创建一段管道，再用 `GetFeatureAtLocation` 定位直管特征，最后进行端口到分支布线：

```vb
' 第一步：点到点布线创建初始管段
Dim oFromPos As New Position(oPosition.X - 5, oPosition.Y - 4, oPosition.Z)
Dim oToPos As New Position(oPosition.X - 5, oPosition.Y + 4, oPosition.Z)
oPipeRun.RouteBetweenPointAndPoint(oFromPos, oToPos, oPipeEndFeature)

' 第二步：定位直管特征
Dim oFeature As RouteFeature = Nothing
Dim oFeatPos As Position = Nothing
oPipeRun.GetFeatureAtLocation(
    PathFeatureObjectTypes.PathFeatureType_STRAIGHT,
    PathFeatureFunctions.PathFeatureFunction_ROUTE,
    New Position(oPosition.X - 5, oPosition.Y, oPosition.Z),
    oFeatPos, oFeature)

' 第三步：端口到分支布线
oPipeRun.RouteBetweenPortAndBranch(oDistribPort, 
    DirectCast(oFeature, PipeStraightFeature), oFeatPos)
```

## 3. 八种特征插入方法（InsertFeature***）

在布线完成后，可以在管段上插入阀门、法兰、仪表等管件特征。

| # | 方法 | 说明 |
|---|------|------|
| 1 | `InsertFeatureByShortCodeOnFeature` | 通过**短代码**在**直管特征**上插入（如 "Gate Valve"） |
| 2 | `InsertFeatureByShortCodeOnPort` | 通过**短代码**在**端口**上插入 |
| 3 | `InsertFeatureByPartOnFeature` | 通过**目录零件**在**直管特征**上插入 |
| 4 | `InsertFeatureByPartOnPort` | 通过**目录零件**在**端口**上插入 |
| 5 | `InsertFeatureByTagOnFeature` | 通过**标签**在**直管特征**上插入（如 "MSCB3W-1"） |
| 6 | `InsertFeatureByTagOnPort` | 通过**标签**在**端口**上插入 |
| 7 | `InsertSplitFeature` | 插入**分割特征**（如法兰），支持 TAKEDOWN 等分割类型 |
| 8 | `GetFeatureAtLocation` | 获取指定位置的特征（配合插入使用） |

### 3.1 InsertFeatureByShortCodeOnFeature

通过短代码（如 "Gate Valve"、"Flange"）在直管特征上插入管件：

```vb
Dim ShortCode As String = "Gate Valve"
Dim iOptionCode As Integer = 1
Dim MultiSize As String = ""
Dim oNewInsertedFeature As RouteFeature = Nothing
Dim InsertGATEAtPos As New Position(123, 0, 0)
Dim oSecondaryRun As RouteRun = oPipeRun

oPipeRun.InsertFeatureByShortCodeOnFeature(
    ShortCode, iOptionCode, MultiSize, oPipeStraightFeature,
    InsertGATEAtPos, oSecondaryRun, oNewInsertedFeature)

MiddleServiceProvider.TransactionMgr.Commit("Inserted a Gate Valve on the SF.")
```

### 3.2 InsertFeatureByShortCodeOnPort

在设备端口上通过短代码插入管件：

```vb
Dim ShortCode As String = "Gate Valve"
Dim iOptionCode As Integer = 1
Dim sMultiSize As String = ""
Dim oNewInsertedFeature As RouteFeature = Nothing

oPipeRun.InsertFeatureByShortCodeOnPort(
    ShortCode, iOptionCode, sMultiSize, oDistribPort, oPipeRun, oNewInsertedFeature)
```

### 3.3 InsertFeatureByPartOnFeature

使用 `CatalogBaseHelper.GetPart` 从目录获取零件后插入：

```vb
' 从目录获取零件
Dim oCatBaseHlpr As New CatalogBaseHelper(oCatConn)
Dim oSOMCatalogPart As IPart = oCatBaseHlpr.GetPart("CIBalloonInstrument4")
Dim oPart As Part = DirectCast(oSOMCatalogPart, Part)

' 在末端特征上插入
Dim oInsertedFeature As RouteFeature = Nothing
oPipeRun.InsertFeatureByPartOnFeature(
    oPart, oPipeEndFeature, oToPoint, oPipeRun, oInsertedFeature)
```

### 3.4 InsertFeatureByPartOnPort

在端口上通过目录零件插入：

```vb
oPipeRun.InsertFeatureByPartOnPort(oPart, oDistribPort, oPipeRun, oNewInsertedFeature)
```

### 3.5 InsertFeatureByTagOnFeature

通过标签（如仪表标签 "MSCB3W-1"）在直管特征上插入：

```vb
Dim sTag As String = "MSCB3W-1"
Dim MultiSize As String = "100x80x50"
Dim oNewInsertedFeature As RouteFeature = Nothing
Dim oSecondaryRun As RouteRun = oPipeRun

oPipeRun.InsertFeatureByTagOnFeature(
    sTag, MultiSize, oPipeEndFeature, oToPoint, oSecondaryRun, oNewInsertedFeature)
```

### 3.6 InsertFeatureByTagOnPort

在端口上通过标签插入：

```vb
Dim sTag As String = "VG3"
Dim sMultiSize As String = ""

oPipeRun.InsertFeatureByTagOnPort(sTag, sMultiSize, oDistribPort, oPipeRun, oNewInsertedFeature)
```

### 3.7 InsertSplitFeature

插入分割特征（如法兰），需指定分割类型 `SplitTypes`：

```vb
Dim sShortCode As String = "Flange"
Dim iOptionCode As Integer = 1
Dim eSplitType As SplitTypes = SplitTypes.TAKEDOWN
Dim oRF As RouteFeature = Nothing

oPipeRun.InsertSplitFeature(sShortCode, iOptionCode, eSplitType, oPipeSF, oTestPos, oRF)
```

### 3.8 GetFeatureAtLocation

按特征类型和功能在指定位置查找特征。使用 `PathFeatureObjectTypes` 和 `PathFeatureFunctions` 两个枚举：

```vb
oPipeRun.GetFeatureAtLocation(
    PathFeatureObjectTypes.PathFeatureType_STRAIGHT,
    PathFeatureFunctions.PathFeatureFunction_ROUTE,
    oAtPosition, oFeaturePosition, oFeature)
```

**PathFeatureObjectTypes 枚举值：**

| 枚举值 | 说明 |
|--------|------|
| `PathFeatureType_ALONG` | 沿线特征 |
| `PathFeatureType_BRANCH` | 分支特征 |
| `PathFeatureType_END` | 末端特征 |
| `PathFeatureType_ENTRYEXIT` | 出入口特征 |
| `PathFeatureType_SEGMENT` | 段特征 |
| `PathFeatureType_STRAIGHT` | 直管特征 |
| `PathFeatureType_SURFACE` | 表面特征 |
| `PathFeatureType_TRANSITION` | 过渡特征 |
| `PathFeatureType_TURN` | 转弯特征 |

**PathFeatureFunctions 枚举值：**

| 枚举值 | 说明 |
|--------|------|
| `PathFeatureFunction_ROUTE` | 布线功能 |
| `PathFeatureFunction_SPLIT` | 分割功能 |
| `PathFeatureFunction_BRANCHPARENT` | 分支父级 |
| `PathFeatureFunction_COMPONENT` | 组件 |
| `PathFeatureFunction_MATING` | 配合 |
| `PathFeatureFunction_ENDLEG` | 末端管腿 |
| `PathFeatureFunction_DEFAULTSPLIT` | 默认分割 |
| `PathFeatureFunction_PLANNINGSPLIT` | 规划分割 |
| `PathFeatureFunction_TAKEDOWN` | 拆解 |

## 4. 属性操作模板

### 4.1 NPD（公称直径）读写

```vb
' 获取 NPD
Dim oNPD As NominalDiameter = oPipeRun.NPD

' 修改 NPD
oNPD.Size = 20
oNPD.Units = "in"
oPipeRun.NPD = oNPD
```

### 4.2 Specification（管道规范）

```vb
Dim oPipeSpec As PipeSpec = TryCast(oPipeRun.Specification, PipeSpec)
```

### 4.3 InsulationSpec（绝缘规范）— IAllowableSpecs 模式

通过 `IAllowableSpecs` 接口从 Pipeline 获取可用规范列表，再筛选绝缘规范：

```vb
' 从 Pipeline 获取所有可用规范
Dim oPipelineSpecs As IAllowableSpecs = TryCast(oPipeline, IAllowableSpecs)
Dim oSpecCollection As ReadOnlyCollection(Of SpecificationBase) = oPipelineSpecs.AllowableSpecs

' 筛选绝缘规范
Dim oPipeInsulationSpec1 As PipeInsulationSpec = Nothing
For Each oSpec As BusinessObject In oSpecCollection
    If oSpec.SupportsInterface("IJPipeInsulationSpec") Then
        oPipeInsulationSpec1 = TryCast(oSpec, PipeInsulationSpec)
        Exit For
    End If
Next

' 设置绝缘规范（含温度）
oPipeRun.SetInsulationSpec(oPipeInsulationSpec1, 310.928)

' 移除绝缘规范
oPipeRun.SetInsulationSpec(Nothing)
```

### 4.4 SetUserDefinedInsulation

自定义绝缘设置（需指定用途和材料的代码列表值）：

```vb
Dim iPurpose As Integer = 1    ' InsulationPurpose codelist
Dim iMaterial As Integer = 1   ' InsulationMaterial codelist
Dim dThickness As Double = 0.5
Dim dTemperature As Double = 150

Try
    oPipeRun.SetUserDefinedInsulation(iPurpose, iMaterial, dThickness, dTemperature)
Catch ex As RteIncorrectPurposeCodeException
    ' 用途代码无效
Catch ex As RteIncorrectMaterialCodeException
    ' 材料代码无效
End Try
```

### 4.5 Features / Parts 集合

```vb
' 获取特征集合
Dim oFeaturesCol As ReadOnlyCollection(Of RouteFeature) = oPipeRun.Features
Dim iFeatCount As Integer = oFeaturesCol.Count

' 获取零件集合
Dim oPartsCol As ReadOnlyCollection(Of RoutePart) = oPipeRun.Parts
Dim iCount As Integer = oPartsCol.Count
```

### 4.6 RangeBox（包围盒）

```vb
Dim oRunRangeBox As RangeBox = oPipeRun.Range
```

## 5. PipeFeature 绝缘规范操作

`PipeFeature.vb` 中的 `PipeFeatureExample` 类演示在 `PipeStraightFeature`（直管特征）上操作绝缘规范。这与 PipeRun 级别的绝缘操作不同——此处是针对**单个特征**的绝缘控制。

### 操作流程

```vb
' 定位直管特征
oPipeRun.GetFeatureAtLocation(
    PathFeatureObjectTypes.PathFeatureType_STRAIGHT,
    PathFeatureFunctions.PathFeatureFunction_ROUTE,
    New Position(225, 0, 0), oSFPosition, oRouteFeat)

Dim oPipeStraightFeat As PipeStraightFeature = DirectCast(oRouteFeat, PipeStraightFeature)

' 获取绝缘规范
Dim oPipeInsulationSpec As PipeInsulationSpec = oPipeStraightFeat.InsulationSpec

' 设置绝缘规范（含温度）
oPipeStraightFeat.SetInsulationSpec(oPipeInsulationSpec1, 500)
oTransactionMgr.Commit("Set InsulationSpec on the Feature")

' 设置绝缘规范（使用已有温度）
oPipeStraightFeat.SetInsulationSpec(oPipeInsulationSpec2)
oTransactionMgr.Commit("Set Insulation Spec on Feature")

' 移除绝缘规范
oPipeStraightFeat.SetInsulationSpec(Nothing)
oTransactionMgr.Commit("Remove Insulation Spec")
```

> **注意**：`SetInsulationSpec` 有两个重载——一个接受 `(spec, temperature)`，另一个仅接受 `(spec)` 并复用特征上已有的温度值。

## 6. RoutePart 操作

`RoutePart.vb` 中的 `RoutePartExample` 类演示路由零件的属性操作。

### 6.1 IsBasePart 转换（MatingPart → BasePart）

- **BasePart（基准件）**：管段中的主要零件，不会被移动或删除
- **MatingPart（配合件）**：附加的配合零件，可以被替换或删除

通过 `IsBasePart = True` 可将配合件转换为基准件：

```vb
' 获取零件集合
Dim oRtePartCol As ReadOnlyCollection(Of RoutePart) = oPipeRun.Parts

' 遍历查找配合件
Dim oPartToBeModified As RoutePart = Nothing
For Each oPart As RoutePart In oRtePartCol
    If oPart.SupportsInterface("IJRteCompOccur") Then
        If oPart.IsBasePart = False Then
            oPartToBeModified = oPart
            Exit For
        End If
    End If
Next

' 转换为基准件
oPartToBeModified.IsBasePart = True
oTransactionMgr.Commit("Convert MatingPart to BasePart")
```

### 6.2 Features 集合

获取 RoutePart 关联的特征集合：

```vb
Dim oRoutePartCol As ReadOnlyCollection(Of RoutePart) = oPipeRun.Parts
Dim oRtePart As RoutePart = oRoutePartCol(0)
Dim oPartFeatCol As ReadOnlyCollection(Of RouteFeature) = oRtePart.Features
```

### 6.3 Range（包围盒）

```vb
Dim oPartRangeBox As RangeBox = oRtePart.Range
```

## 7. 完整工作流图

```
┌─────────────────────────────────────────────────────────────────┐
│                    管道布线完整工作流                              │
└─────────────────────────────────────────────────────────────────┘

  ① 创建 Pipeline（管道线系统）
     │  NewPipeLineSystem("MyPipeline")
     │     → oRootSystem → Pipeline → SetPropertyValue(Name)
     ▼
  ② 创建 PipeRun（管道段）
     │  NewPipeRun(oPipeline, NPD, Unit, Spec, Name, Temp, Pressure)
     │     → PipeRun → SetPropertyValue(Name/Temp/Pressure)
     ▼
  ③ 布线 RouteBetween***（10种方法）
     │  ┌── 点到点 / 端口到端口 / 端口到分支
     │  ├── 端口到点 / 端口到末端特征
     │  ├── 分支到分支 / 分支到点 / 分支到末端特征
     │  └── 末端特征到点 / 末端特征到末端特征
     ▼
  ④ 插入特征 InsertFeature***（8种方法）
     │  ┌── ShortCode → OnFeature / OnPort
     │  ├── Part     → OnFeature / OnPort（CatalogBaseHelper.GetPart）
     │  ├── Tag      → OnFeature / OnPort
     │  └── InsertSplitFeature（TAKEDOWN）
     ▼
  ⑤ 设置属性 SetProperties
     │  ┌── NPD（公称直径）
     │  ├── Specification（管道规范）
     │  ├── InsulationSpec（绝缘规范，IAllowableSpecs 模式）
     │  └── SetUserDefinedInsulation（自定义绝缘）
     ▼
  ⑥ 获取集合 → 操作子对象
     │
     ├── Parts 集合 → RoutePart 操作
     │     ├── IsBasePart（MatingPart ↔ BasePart 转换）
     │     ├── Features（零件关联特征）
     │     └── Range（包围盒）
     │
     └── Features 集合 → PipeFeature 操作
           ├── GetInsulationSpec / SetInsulationSpec / RemoveInsulationSpec
           └── Range（特征包围盒）
```

### 辅助函数：从设备获取管道端口

```vb
Private Sub GetPipePortFromEquipment(oOrigin As Position, sPortName As String, 
    ByRef oDistribPort As DistributionPort)
    ' 创建设备并设置原点
    Dim oEqp As Equipment = New Equipment("PUMP 001A-E", Nothing)
    Dim oEqpCS As ILocalCoordinateSystem = DirectCast(oEqp, ILocalCoordinateSystem)
    oEqpCS.Origin = oOrigin
    
    ' 获取管道端口
    Dim oReadCollPorts As ReadOnlyCollection(Of IPort) = oEqp.GetConnectablePorts(PortType.Piping)
    For Each oPort As IPort In oReadCollPorts
        If oPort.ToString().ToUpper().Equals(sPortName.ToUpper()) Then
            oDistribPort = DirectCast(oPort, DistributionPort)
            Exit For
        End If
    Next
End Sub
```
