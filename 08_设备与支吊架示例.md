### 1. 四种设备创建方式（含代码）

#### 1.1 方式一：按零件号创建（Under Project Root）

最简单的方式，直接指定零件号，`Nothing` 表示放在项目根节点下：

```vb
Private Sub PlaceEquipment()
    Dim oTransactionMgr As TransactionManager = MiddleServiceProvider.TransactionMgr
    Dim oEqp As Equipment = Nothing

    ' 按零件号创建设备，Nothing 表示放在项目根节点下
    oEqp = New Equipment("PUMP 001A-E", Nothing)

    If oEqp IsNot Nothing Then
        ' 设置设备位置
        Dim oEqpCS As ILocalCoordinateSystem = DirectCast(oEqp, ILocalCoordinateSystem)
        Dim oPos As New Position()
        oPos.X = 1.0R
        oPos.Y = 2.0R
        oPos.Z = 3.0R
        oEqpCS.origin = oPos
        oTransactionMgr.Commit("Place Equipment")
    End If
End Sub
```

#### 1.2 方式二：按 Part 对象创建

先通过 `CatalogBaseHelper.GetPart` 获取 Part 对象，再用 Part 创建设备：

```vb
Private Sub PlaceEquipmentGivenPart()
    Dim oSiteMgr As SiteManager = MiddleServiceProvider.SiteMgr
    Dim oPlant As Plant = oSiteMgr.ActiveSite.ActivePlant
    Dim oCatalog As Catalog = oPlant.PlantCatalog

    ' 通过目录助手获取 Part 对象
    Dim oCatalogBaseHelper As New CatalogBaseHelper(oCatalog)
    Dim oPart As Part = DirectCast(oCatalogBaseHelper.GetPart("PUMP 001A-E"), Part)

    If oPart IsNot Nothing Then
        ' 用 Part 对象创建设备
        Dim oEqp As New Equipment(oPart, Nothing)

        ' 设置位置
        Dim oPos As New Position(2.5, -1.5, 1.5)
        Dim oDesignEqpCS As ILocalCoordinateSystem = DirectCast(oEqp, ILocalCoordinateSystem)
        oDesignEqpCS.origin = oPos
        oTransactionMgr.Commit("Place Design Equipment")
    End If
End Sub
```

#### 1.3 方式三：在父系统下创建

先创建 `EquipmentSystem` 作为父系统，再在其下创建设备：

```vb
Private Sub PlaceEquipmentUnderParentSystem()
    Dim oModel As Model = oPlant.PlantModel
    Dim oRootSystem As BusinessObject = DirectCast(oModel.RootSystem, BusinessObject)

    ' 创建父系统
    Dim oParentSystem As New EquipmentSystem(oRootSystem)
    oTransactionMgr.Commit("create parent EquipmentSytem")

    ' 命名父系统
    Dim oParentSystemBO As BusinessObject = DirectCast(oParentSystem, BusinessObject)
    oParentSystemBO.SetPropertyValue("Equipment_SOMEqupmentWithSystem", "IJNamedItem", "Name")
    oTransactionMgr.Commit("naming parentEquipmentSytem")

    ' 在父系统下创建设备
    Dim oEqp As Equipment = New Equipment("PUMP 001A-E", DirectCast(oParentSystemBO, ISystem))

    ' 设置位置
    Dim oEqpCS As ILocalCoordinateSystem = DirectCast(oEqp, ILocalCoordinateSystem)
    Dim oPos As New Position(1, 2, 3)
    oEqpCS.origin = oPos
    oTransactionMgr.Commit("Place Equipment")
End Sub
```

#### 1.4 方式四：创建 Design Equipment（按 PartClass）

通过 `PartClass`（而非具体 Part）创建设计设备，适用于尚未确定具体型号的场景：

```vb
Private Sub PlaceDesignEquipment()
    Dim oCatalog As Catalog = oPlant.PlantCatalog
    Dim oRootSystem As BusinessObject = DirectCast(oModel.RootSystem, BusinessObject)

    ' 创建父系统
    Dim oParentSystem As New EquipmentSystem(oRootSystem)
    oTransactionMgr.Commit("create EquipmentSystem")

    ' 获取 PartClass（零件类别）
    Dim oCatalogBaseHelper As New CatalogBaseHelper(oCatalog)
    Dim oPartClass As PartClass = DirectCast(oCatalogBaseHelper.GetPartClass("PumpAsm"), PartClass)

    ' 用 PartClass 创建设计设备
    Dim oDesignEqp As New Equipment(oPartClass, DirectCast(oParentSystemBO, ISystem))

    ' 设置位置
    Dim oPos As New Position(2.5, -1.5, 1.5)
    Dim oDesignEqpCS As ILocalCoordinateSystem = DirectCast(oDesignEqp, ILocalCoordinateSystem)
    oDesignEqpCS.origin = oPos
    oTransactionMgr.Commit("Place Design Equipment")
End Sub
```

#### 四种创建方式对比

| 方式 | 构造签名 | 适用场景 |
|------|----------|----------|
| 按零件号 | `New Equipment(partNumber, Nothing)` | 快速放置，根节点下 |
| 按 Part 对象 | `New Equipment(oPart, Nothing)` | 已获取 Part 对象 |
| 父系统下 | `New Equipment(partNumber, parentSystem)` | 需要组织到系统中 |
| Design Equipment | `New Equipment(partClass, parentSystem)` | 设计阶段，按类别创建 |

### 2. 位置设置统一模式

所有设备和形状的位置都通过 `ILocalCoordinateSystem.origin` 设置：

```vb
' 设备位置设置（通用模式）
Dim oEqpCS As ILocalCoordinateSystem = DirectCast(oEqp, ILocalCoordinateSystem)
Dim oPos As New Position()
oPos.X = 1.0R
oPos.Y = 2.0R
oPos.Z = 3.0R

```

> **关键接口**: `ILocalCoordinateSystem` — 设备、形状、FoundationPort 都实现此接口，通过 `origin` 属性设置位置。

### 3. 六种端口创建（含构造签名）

#### 3.1 PipeNozzle — 管道管口

两种构造方式：基础构造器（需手动设置位置/方向）和完整构造器（带位置和方向）：

```vb
' 方式一：基础构造器，创建后手动设置位置和方向
Dim oPort As PipeNozzle = New PipeNozzle("PUMP 001A-E", False, 1, oEqp)
' 参数: partNumber, isGeneric, index, parentEquipment

If oPort IsNot Nothing Then
    Dim oPortCS As IDistributionPort = DirectCast(oPort, IDistributionPort)
    Dim oPos As New Position(1, 3, 3)
    Dim oDirVec As New Vector(0, 0, 1)
    oPortCS.Location = oPos          ' 设置位置
    oPort.NormalVector = oDirVec     ' 设置法向量（方向）
    oTransactionMgr.Commit("Place Nozzle1")
End If

' 方式二：完整构造器，直接指定位置和方向
Dim oVect As New Vector(1, 0, 0)
Dim oPos1 As New Position(1, 2, 3)
Dim oConn As SP3DConnection = oEqp.DBConnection

Dim opipeNozzle3 As New PipeNozzle("PUMP 001A-E", oConn, False, 1, oPos1, oVect, 0.2, False)
' 参数: partNumber, connection, isGeneric, index, position, direction, rotation, isMirrored
opipeNozzle3.SystemParent = oEqp
```

#### 3.2 HvacPort — 暖通端口

```vb
' 基础构造器
Dim oHvacPort1 As New HvacPort("VVADA 001A-E", False, 1, oEqp)

Dim oPortCS As IDistributionPort = DirectCast(oHvacPort1, IDistributionPort)
oPortCS.Location = New Position(1, 3, 3)
```

#### 3.6 FoundationPort — 基础端口

FoundationPort 与其他端口不同，直接使用 `ILocalCoordinateSystem.origin` 设置位置：

```vb
Dim oFoundationPort1 As New FoundationPort("PUMP 001A-E", 3, oEqp)
' 参数: partNumber, index, parentEquipment

If oFoundationPort1 IsNot Nothing Then
    Dim oPos As New Position(2, 2, 3)
    oFoundationPort1.origin = oPos   ' 直接使用 origin 属性（ILocalCoordinateSystem）
    oTransactionMgr.Commit("Place oFoundationPort1")
End If
```

#### 六种端口对比表

| 端口类型 | 基础构造器 | 位置设置方式 |
|----------|-----------|-------------|
| `PipeNozzle` | `New PipeNozzle(partNum, isGeneric, index, parent)` | `IDistributionPort.Location` + `NormalVector` |
| `HvacPort` | `New HvacPort(partNum, isGeneric, index, parent)` | `IDistributionPort.Location` + `NormalVector` |
| `CablePort` | `New CablePort(partNum, index, parentSystem)` | `IDistributionPort.Location` + `NormalVector` |
| `CableTrayPort` | `New CableTrayPort(partNum, index, parentSystem)` | `IDistributionPort.Location` + `NormalVector` |
| `ConduitPort` | `New ConduitPort(partNum, index, parentSystem)` | `IDistributionPort.Location` + `NormalVector` |
| `FoundationPort` | `New FoundationPort(partNum, index, parentEquipment)` | `ILocalCoordinateSystem.origin` |

> **两种位置模式**: 前5种端口使用 `IDistributionPort.Location` + `NormalVector` 设置位置和方向；FoundationPort 使用 `ILocalCoordinateSystem.origin` 直接设置原点。

#### 4.2 PrismaticShape — 棱柱形状

通过 `Sketch3D`（路径）和 `SketchCrossSection`（截面）创建拉伸体：

```vb
Private Sub PlacePrismaticShape()
    Dim oModel As Model = oPlant.PlantModel
    Dim oEqp As Equipment = New Equipment("PUMP 001A-E", Nothing)
    ' ... 设置设备位置 ...

    ' 步骤1: 创建路径（Sketch3D）
    Dim oCollection As New Collection(Of SketchPoint)()
    Dim oSketchPoint1 As New SketchPoint(0, 0, 0)
    Dim oSketchPoint2 As New SketchPoint(0, 0, 4)
    Dim oSketchPoint3 As New SketchPoint(0, 4, 4)

    ' 创建线特征连接草图点
    Dim oLine As LineFeature = oSketchPoint1.CreateLineFeature(oSketchPoint2)
    oLine = oSketchPoint2.CreateLineFeature(oSketchPoint3)

    oCollection.Add(oSketchPoint1)
    oCollection.Add(oSketchPoint2)
    oCollection.Add(oSketchPoint3)

    Dim oPath As New Sketch3D(oModel)
    oPath.SetSketchPoints(oCollection)

    ' 步骤2: 创建截面（SketchCrossSection）
    Dim oSketchCrossSection As New SketchCrossSection(oModel)
    Dim oCataloBaseHelper As New CatalogBaseHelper()
    Dim oPartCs As Part = DirectCast(oCataloBaseHelper.GetPart("SKETCH_CROSS_SECTION_02"), Part)
    oSketchCrossSection.CrossSectionDefinition = oPartCs

    ' 步骤3: 创建棱柱形状
    Dim oPrismaticShape As PrismaticShape = New PrismaticShape(
        "PrismaticShape", oSketchCrossSection, oPath, oEqp)
    ' 参数: partNumber, crossSection, path, parentEquipment

    ' 设置位置（移动草图点的坐标）
    Dim oPosition As New Position(2, 0, 0)
    oPrismaticShape.origin = oPosition
    oTransactionMgr.Commit("Place prismaticShape")
End Sub
```

> **PrismaticShape 构造器变体**: 可用 Part 对象替代零件号：`New PrismaticShape(oPart, crossSection, path, parent)`

### 5. 端口约束

在两个设备的端口之间创建约束连接：

```vb
Private Sub CreateConstraint()
    ' 创建两个设备
    Dim oEqp1 As Equipment = New Equipment("PUMP 001A-E", Nothing)
    Dim oEqp2 As Equipment = New Equipment("PUMP 001A-E", Nothing)
    ' ... 设置位置 ...

    ' 获取两个设备的管道端口集合
    Dim oPort1Col As ReadOnlyCollection(Of IPort) = 
        DirectCast(oEqp1, IConnectable).GetConnectablePorts(PortType.Piping)
    Dim oPort2Col As ReadOnlyCollection(Of IPort) = 
        DirectCast(oEqp2, IConnectable).GetConnectablePorts(PortType.Piping)

    ' 在两个端口之间创建约束
    If (oPort1Col.Count > 0) And (oPort2Col.Count > 0) Then
        Dim oConstraint As New Constraint(oPort1Col(0), oPort2Col(0))
        oTransactionMgr.Commit("test Constraint")
    End If
End Sub
```

> **关键接口**: `IConnectable.GetConnectablePorts(PortType)` — 按端口类型（Piping/Hvac/Cable等）获取设备的端口集合。

### 1. PipeSupport 完整创建工作流

管道支吊架的创建遵循标准的10步工作流：

```vb
Private Sub CreatePipeSupportExamples()
    ' 步骤1: 获取根系统
    Dim oRootObject As BusinessObject = _
        TryCast(MiddleServiceProvider.SiteMgr.ActiveSite.ActivePlant.PlantModel.RootSystem, BusinessObject)
    Dim oSystem As ISystem = oRootObject

    ' 步骤2: 准备被支撑对象集合和支撑对象集合
    Dim oRouteFeature As RouteFeature = Nothing  ' 被支撑的管道特征
    Dim oSupportingObjColl As New Collection(Of BusinessObject)()   ' 支撑结构集合
    Dim oSupportedObjColl As New Collection(Of RouteFeature)()      ' 被支撑对象集合
    oSupportedObjColl.Add(oRouteFeature)

    ' 步骤3: 获取目录结构助手
    Dim oCatConn As SP3DConnection = MiddleServiceProvider.SiteMgr.ActiveSite.ActivePlant.PlantCatalog
    Dim oCatStruHlpr As New CatalogStructHelper(oCatConn)

    ' 步骤4: 获取截面
    Dim oCrossSection As CrossSection = oCatStruHlpr.GetCrossSection("AISC-LRFD-3.1", "W", "W8x10")

    ' 步骤5: 获取材料
    Dim oMaterial As Material = oCatStruHlpr.GetMaterial("Steel - Carbon", "A")

    ' 步骤6: 创建 MemberPart（支撑结构件）
    Dim lCategory As Integer = MemberType.Beam
    Dim lCardinalPoint As Integer = 8
    Dim oStartPosition As New Position(50, 10, 20)
    Dim oEndPosition As New Position(60, 10, 20)

    Dim oMemberPart1 As New MemberPart(oStructSystem, oStartPosition, oEndPosition, 
        oCrossSection, oMaterial, lCategory, lCardinalPoint, 0, False)
    oSupportingObjColl.Add(oMemberPart1)

    ' 步骤7: 创建 PipeSupport（基础构造器）
    Dim oPipeSupport As New PipeSupport(oSystem, oSupportedObjColl, oSupportingObjColl)

    ' 步骤8: 设置支撑定义
    Dim oSupUtil As New SupUtilitis
    Dim oPart As Part = oSupUtil.CreatePart("Assy_FR_IT_LS_7")
    oPipeSupport.SupportDefinition = oPart

    ' 步骤9: 带位置的构造器变体
    Dim oPos As New Position(55, 10, 20)
    Dim oPipeSupport2 As New PipeSupport(oSystem, oSupportedObjColl, oSupportingObjColl, oPos)
    oPart = oSupUtil.CreatePart("Assy_FR_IT_LS_2")
    oPipeSupport2.SupportDefinition = oPart

    ' 步骤10: 提交（每个构造后都需 Commit）
End Sub
```

### 3. Port 操作

`PortExample` 演示支吊架端口的创建和方向操作。

#### 3.1 创建端口（两种构造器）

```vb
Private Sub CreatePortExamples()
    Dim oRootObject As BusinessObject = _
        TryCast(MiddleServiceProvider.SiteMgr.ActiveSite.ActivePlant.PlantModel.RootSystem, BusinessObject)
    Dim oPart As Part = Nothing

    ' 简单构造器
    Dim oPort As New Port(oRootObject.DBConnection, oPart, "TestPort")
    ' 参数: connection, part, name

    ' 带位置和方向的构造器
    Dim oPortOrigin As New Position(0, 80, 0)
    Dim oXDirVector As New Vector(1, 80, 0)
    Dim oZDirVector As New Vector(0, 80, 1)
    Dim oPort2 As New Port(oRootObject.DBConnection, oPart, "TestPort2", 
        oPortOrigin, oXDirVector, oZDirVector)
    ' 参数: connection, part, name, origin, xAxisVector, zAxisVector
End Sub
```

### 4. 组件获取

`SupportComponentExample` 演示如何遍历支吊架的子组件并进行类型判断：

```vb
Private Sub GetSupportComponentExamples()
    Dim oPipeSup As PipeSupport = Nothing  ' 已创建的支吊架

    ' 获取所有子组件
    Dim oSupComps As ReadOnlyCollection(Of ISystemChild) = oPipeSup.SystemChildren

    ' 遍历并按类型分类
    Dim oStdSupComp As StandardComponent = Nothing
    Dim oLogSupComp As LogicalComponent = Nothing
    Dim oConSupComp As ConnectionComponent = Nothing
    Dim oHgrConn As HgrConnection = Nothing

    For Each oSupComp In oSupComps
        If TypeOf (oSupComp) Is StandardComponent Then
            oStdSupComp = oSupComp        ' 标准组件
        End If
        If TypeOf (oSupComp) Is LogicalComponent Then
            oLogSupComp = oSupComp        ' 逻辑组件
        End If
        If TypeOf (oSupComp) Is ConnectionComponent Then
            oConSupComp = oSupComp        ' 连接组件
        End If
        If TypeOf (oSupComp) Is HgrConnection Then
            oHgrConn = oSupComp           ' 支吊架连接
        End If
    Next

    ' 获取组件的父级
    Dim oSysParent As ISystem = oStdSupComp.SystemParent     ' 系统父级
    Dim oAssParent As IAssembly = oStdSupComp.AssemblyParent ' 装配体父级
End Sub
```

|----------|------|
| `StandardComponent` | 标准组件（实体零件） |
| `LogicalComponent` | 逻辑组件（非实体，如逻辑分组） |
| `ConnectionComponent` | 连接组件（连接关系） |
| `HgrConnection` | 支吊架专用连接 |

## 三、开发要点总结

### Equipment 开发要点

1. **四种创建方式选择**: 快速放置用零件号，需要系统组织用 ParentSystem，设计阶段用 PartClass
2. **位置统一模式**: 所有设备通过 `ILocalCoordinateSystem.origin` 设置位置
3. **端口两种位置模式**: 大多数端口用 `IDistributionPort.Location` + `NormalVector`；FoundationPort 用 `origin`
4. **完整构造器**: PipeNozzle 和 HvacPort 支持带位置和方向的完整构造器，无需后续单独设置
5. **PrismaticShape**: 需要 Sketch3D（路径）+ SketchCrossSection（截面）两部分，通过线特征连接草图点
6. **端口约束**: 通过 `IConnectable.GetConnectablePorts(PortType)` 获取端口，再用 `New Constraint` 连接

### HangersAndSupports 开发要点

1. **统一创建模式**: 六种支吊架都遵循 `New XxxSupport(system, supportedColl, supportingColl)` 模式
2. **两个集合**: `supportedColl`（被支撑对象，如管道 RouteFeature）和 `supportingColl`（支撑结构，如 MemberPart）
3. **SupportDefinition**: 创建后必须通过 `SupUtilitis.CreatePart` 获取支撑定义零件并赋值
4. **DesignSupport 特殊**: 可从已有 PipeSupport 转换，支持两种转换构造器
5. **端口方向**: Port 通过两个向量（X轴和Z轴）定义方向，可用 `GetOrientation`/`SetOrientation` 操作
6. **组件遍历**: `SystemChildren` 返回混合类型的集合，需用 `TypeOf ... Is ...` 进行类型判断
