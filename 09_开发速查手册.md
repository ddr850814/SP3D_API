## 四、位置与方向设置模式

**模式一：ILocalCoordinateSystem（设备、形状、基础端口等）**

适用于 Equipment、GenericShape、FoundationPort、PrismaticShape 等对象：

```vb
DirectCast(obj, ILocalCoordinateSystem).origin = position
```

**模式二：IDistributionPort（管口、HVAC、电缆等端口）**

适用于 PipeNozzle、HvacPort、CablePort 等端口对象：

```vb
DirectCast(port, IDistributionPort).Location = position
DirectCast(port, IDistributionPort).NormalVector = vector
```

## 六、几何对象统一生命周期

Smart3D 中所有几何对象（Line3d、Circle3d、Projection3d 等）遵循统一的生命周期管理流程：

```vb
' 1. 构造
Dim oLine As Line3d = New Line3d(startPos, endPos)

' 2. 设属性（可选）
oLine.Color = Color.Red

' 3. DefineBy*** 重定义（可选）
oLine.DefineByPoints(startPos, endPos)

' 4. 边界操作（裁剪、延伸等）
oLine.Trim(trimPoint)

' 5. GetExtrema 获取范围
Dim minPt As Position, maxPt As Position
oLine.GetExtrema(minPt, maxPt)

' 6. Transform 变换
oLine.Transform(oMatrix)

' 7. Delete 释放
oLine.Delete()
oLine = Nothing
```

## 七、命令开发三种基类

| 基类 | EnableUIFlags | Modal | Suspendable | 生命周期方法 |
| --- | --- | --- | --- | --- |
| BaseGraphicCommand | 可配置 | False | True | OnStart / OnStop / OnSuspend / OnResume |
| BaseStepCommand | 可配置 | False | True | OnStart / OnStop / OnSuspend / OnResume |
| BaseModalCommand | 默认 | True | False | OnStart / OnStop |

**说明：**
- `BaseGraphicCommand`：用于图形交互命令，支持拾取、动态预览。
- `BaseStepCommand`：用于分步向导式命令，支持多步骤输入。
- `BaseModalCommand`：模态命令，执行期间阻塞其他操作，不支持挂起/恢复。

```vb
Public Class MyCommand
    Inherits BaseGraphicCommand

    End Sub

    End Sub

    Public Overrides Sub OnSuspend()
        ' 挂起处理
    End Sub

    Public Overrides Sub OnResume()
        ' 恢复处理
    End Sub
End Class
```

## 十二、常见陷阱与注意事项

1. **集合索引差异**：SP3DConnections 从 1 开始，BOCollection 从 0 开始。混用时极易出错，需格外留意。

2. **事务必须提交**：所有写操作后必须调用 `TransactionMgr.Commit()`，否则数据不持久化，程序结束后修改将丢失。

3. **origin vs Location**：
   - Equipment / GenericShape / PrismaticShape 等使用 `ILocalCoordinateSystem.origin` 设置位置。
   - PipeNozzle / HvacPort / CablePort 等端口使用 `IDistributionPort.Location` 设置位置。

4. **矩阵操作是 in-place**：`Rotate` / `Scale` / `Translate` / `MultiplyMatrix` / `Invert` 都修改自身矩阵，不返回新对象。若需保留原矩阵，请先 `Clone`。

5. **几何对象删除后引用置 Nothing**：调用 `Delete()` 后应立即将变量置为 `Nothing`，避免悬挂引用导致后续访问异常。

6. **保温厚度为零时创建微小几何体**：当保温层厚度为 0 时，应使用微小值（如 `0.001`）替代，避免几何计算报错。

7. **SketchPoint.CreateXxxFeature 失败会抛异常**：所有 `CreateXxxFeature` 方法在条件不满足时会抛出异常，必须使用 `Try/Catch` 包裹。

8. **ChamferFeature / BendFeature 需要两条相邻曲线**：倒角和弯曲特征要求输入的两条曲线必须相邻，否则创建失败。

9. **SaveAs 后需要 Abort 事务**：调用 `SaveAs` 导出图纸后，应 `Abort` 当前事务，避免在数据库中产生不必要的脏数据。

10. **RestoreSite 可能抛 CmnOutOfDateSiteDatabaseException**：恢复站点时若数据库版本不匹配会抛此异常，后续需要执行 `MigrateSite` 完成数据库迁移。
