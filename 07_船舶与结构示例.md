## 一、Planning — 规划工作流

### 1. 模型创建模板（CreateModel 完整代码）

`CreateModel()` 方法是整个示例的基础，演示了从零创建船舶结构模型的完整流程。

#### 1.1 创建 StructuralSystem + GridSystem

```vb
' 获取根系统和服务
Dim oTransactionMgr As TransactionManager = MiddleServiceProvider.TransactionMgr
Dim oSiteMgr As SiteManager = MiddleServiceProvider.SiteMgr
Dim oPlant As Plant = oSiteMgr.ActiveSite.ActivePlant
Dim oModel As Model = oPlant.PlantModel
Dim ocatalogDB As Catalog = oPlant.PlantCatalog
Dim catStructHelper As New CatalogStructHelper(ocatalogDB)

' 创建 StructuralSystem（所有板/加强筋/型材的父系统）
Dim rootSystem As ISystem = DirectCast(oModel.RootSystem, ISystem)
Dim structuralSystem As New StructuralSystem(DirectCast(rootSystem, BusinessObject))
structuralSystem.SetUserDefinedName("PlanningSOMTestCommands")

' 创建 GridSystem + CoordinateSystem（Ship 类型坐标系）
Dim gridSystem As New GridSystem(structuralSystem, 
    CoordinateSystem.CoordinateSystemType.Ship, New Position(0.0, 0.0, 0.0))

' 定义矩形网格：GridAxisInputs(start, step, count)
Dim cs As CoordinateSystem = gridSystem.GetGridSystem(
    GridCoordinateTypes.Rectangular,
    New GridAxisInputs(0.0, 1.0, 30),    ' X轴: 起点0, 步长1, 30条
    New GridAxisInputs(0.0, 5.0, 20), False,  ' Y轴: 起点0, 步长5, 20条
    New GridAxisInputs(0.0, 2.0, 20), False)  ' Z轴: 起点0, 步长2, 20条
cs.SetUserDefinedName("CS_Planning")
oTransactionMgr.Commit("CS_Mfg")

' 获取三个轴用于后续创建平面
gridAxisX = gridSystem.GetGridAxis(cs, AxisType.X)
gridAxisY = gridSystem.GetGridAxis(cs, AxisType.Y)
gridAxisZ = gridSystem.GetGridAxis(cs, AxisType.Z)
```

#### 1.2 创建 PlateSystem（板系统）

```vb
' 获取材料和命名类别
Dim material As Material = catStructHelper.GetMaterial("Steel - Carbon", "A")
Dim namingCategory As Integer = StructMarineHelper.GetPlateNamingCategoryCodelistValue(
    PlateType.DeckPlate, "LS")

' 创建 PlateCommonInputs（板的通用输入参数）
Dim plateCommonInputs As New PlateCommonInputs(
    PlateType.DeckPlate,              ' 板类型：甲板板
    material,                          ' 材料
    0.006,                             ' 厚度
    Continuity.Continuous,             ' 连续性
    MoldedDirection.Above,             ' 成型方向
    Tightness.NonTight,                ' 密封性
    namingCategory,                    ' 命名类别
    0)                                 ' 附加参数

' 创建平面定义（通过 Z 轴的平面定位板）
Dim planeDef As New PlaneDefinition(gridAxisZ.GetPlane(10.0))

' 创建边界集合（用网格平面定义板的边界）
Dim boundariesColl As New Collection(Of Object)()
boundariesColl.Add(gridAxisY.GetPlane(0.0))
boundariesColl.Add(gridAxisY.GetPlane(20.0))
boundariesColl.Add(gridAxisX.GetPlane(40.0))
boundariesColl.Add(gridAxisX.GetPlane(75.0))

' 创建板系统
planarDeckPlate = New PlateSystem(structuralSystem, planeDef, boundariesColl, plateCommonInputs)
planarDeckPlate.SetUserDefinedName("PlanarDeckPlate")
oTransactionMgr.Commit("PlanarDeckPlate Commit")

' 细化板系统（生成 PlatePart）
planarDeckPlate.Detail()
oTransactionMgr.Commit("Detailed the Plate Systems")

' 获取板零件
Dim sysPlatePartColl As ReadOnlyCollection(Of PlatePart) = planarDeckPlate.GetPlateParts()
planarDeckPlatePart = sysPlatePartColl(0)
```

#### 1.3 创建 StiffenerSystem（加强筋系统）

```vb
' 获取截面
Dim crossSection As CrossSection = catStructHelper.GetCrossSection(
    "ShipShapes", "TeeBar", "T 100x80")

' 创建 StiffenerCommonInputs
Dim mountingFace As Integer = CInt(StiffenerMountingFaceName.BottomSideOfInferiorFlange)
Dim loadPoint As Integer = 3
Dim profileMountingPoint As Integer = CInt(StiffenerMountingPoint.NoTrim)
Dim projectionMethod As Integer = CInt(LandingCurveProjectionMethod.AlongWeb)

Dim StiffenerCommonInputs As New StiffenerCommonInputs(
    crossSection, material,
    MoldedOrientation.Above, MoldedOrientation.Starboard,
    mountingFace, loadPoint, profileMountingPoint, projectionMethod, namingCategory)

Dim oBoundariesColl As New Collection(Of Object)()

' 通过相交平面创建加强筋
stiffenerSystem1 = New StiffenerSystem(planarDeckPlate, gridAxisY.GetPlane(2.0), 
    StiffenerCommonInputs, oBoundariesColl)
stiffenerSystem1.SetUserDefinedName("StiffenerSystem1")
oTransactionMgr.Commit("StiffenerSystem")

' 细化并获取加强筋零件
stiffenerSystem1.Detail()
Dim sysProfilePartColl As ReadOnlyCollection(Of ProfilePart) = stiffenerSystem1.GetParts()
stiffenerPart1 = DirectCast(sysProfilePartColl(0), StiffenerPart)
```

#### 1.4 创建 MemberSystem（型材系统）

```vb

' 获取型材零件
```


### 3. 块分割与合并

#### 3.1 分割块

```vb
Dim rootBlock As Block = Block.GetRootBlock()

' 收集分割平面
Dim planes As ICollection(Of Object) = New Collection(Of Object)()
planes.Add(gridAxisX.GetPlane(50.0))
planes.Add(gridAxisX.GetPlane(70.0))
planes.Add(gridAxisX.GetPlane(80.0))

' 去重
Dim myArray As Object() = New Object(planes.Count - 1) {}
planes.CopyTo(myArray, 0)
Dim splitters As IEnumerable(Of Object) = myArray.Distinct()

' 执行分割：参数2=是否删除空块, 参数3=是否创建子块
rootBlock.Split(splitters, False, True)
oTransactionMgr.Commit("Block Splitted")
```

#### 3.2 合并块

```vb
Dim rootBlock As Block = Ingr.SP3D.Planning.Middle.Block.GetRootBlock()

' 获取所有子块
Dim blocks As ReadOnlyCollection(Of IAssemblyChild) = 
    rootBlock.GetAssemblyChildren("IJBlock", False)

Dim blockColl As IList(Of Block) = New List(Of Block)
For Each block In blocks
    blockColl.Add(DirectCast(block, Ingr.SP3D.Planning.Middle.Block))
Next

' 合并所有块（参数: 是否删除源块）
Ingr.SP3D.Planning.Middle.Block.Merge(blockColl, True)
```

#### 1.3 核心方法

```csharp
// 初始化：缓存框架传入的参数
public override void Initialize(ViewBase view, ByRuleCache cacheWrapper, string settings)

    this.cache = cacheWrapper;
    this.view = view;
    this.settings = settings;

public override object GetProperty(BusinessObject elementBO)
```

### 2. SampleGraphicRule — 端切符号规则 (SampleEndCutRule.cs)

> **Path**: `ShipDrawings\SOM\Examples\SampleGraphicRule\SampleEndCutRule.cs`
> **Namespace**: `SampleEndCutRules`

#### 2.1 功能说明

DBR（Drawing By Rule）规则实现，根据型材的截面类型、连接类型和端切数据，确定在图纸上绘制端切符号时使用的符号文件路径。

#### 2.2 类结构

```csharp
public class SampleEndCutRule : EndCutRule
{
    private struct Codelist
    {
        public string sTableName;
        public string sNamespace;
    }

    private string m_sSymbolDirectory = "";
    private XmlNode m_xCustomNode = null;
    private Codelist[] m_sCodelists = { ... };  // 4个码表
}
```

#### 2.3 核心工作流

```
GetEndCutSymbolFilePath(bo, location)
  │
  ├── GetProfileInfo(bo, location)       — 提取15维型材信息
  │     ├── [0] ProfileSide (Start/End)
  │     ├── [1] SectionType (截面类型)
  │     ├── [2] ConnectionType (连接类型)
  │     ├── [3-5] PrimaryWebCut (主腹板切割)
  │     ├── [6-8] TopFlangeCut (上翼缘切割)
  │     ├── [9-11] BottomFlangeCut (下翼缘切割)
  │     └── [12-14] SecondaryWebCut (次腹板切割)
  │
  ├── GetXMLSearchPath(profileInfo)      — 构建XPath查询字符串
  │
  └── SelectNode + Symbol                — 从XML匹配节点，返回符号文件路径
```

#### 2.4 Initialize — 加载 XML 规则

```csharp
public override void Initialize(string endcutRulePath)
{
    // 验证文件存在
    m_sSymbolDirectory = System.IO.Directory.GetParent(endcutRulePath).FullName;

    // 加载 XML 规则文件
    XmlDocument xDocument = new XmlDocument();
    xDocument.Load(endcutRulePath);

    // 获取 RULE/CUSTOM 根节点
    m_xCustomNode = xDocument.SelectSingleNode("RULE/CUSTOM");

    // 读取可选的码表配置
    GetCodelists();
}
```

#### 2.5 GetEndCutSymbolFilePath — 核心方法

```csharp
public override string GetEndCutSymbolFilePath(BusinessObject bo, PositionOnObject location)
{
    // 提取15维型材信息
    string[] sProfileInfo = GetProfileInfo(bo, location);

    // 构建 XPath 搜索路径
    string sXMLSearchPath = GetXMLSearchPath(sProfileInfo);

    // 从 XML 匹配节点并获取符号文件名
    XmlNode xInfoNode = m_xCustomNode.SelectSingleNode(sXMLSearchPath);
    if (xInfoNode != null)
    {
        XmlNode xSymbolNode = xInfoNode.SelectSingleNode("Symbol");
        if (xSymbolNode != null)
        {
            symbolFilePath = System.IO.Path.Combine(
                m_sSymbolDirectory, xSymbolNode.InnerText);
        }
    }
    return symbolFilePath;
}
```

#### 2.6 15维型材信息详解

| 索引 | 含义 | 数据来源 |
|------|------|----------|
| 0 | ProfileSide (Start/End) | `location` 参数 |
| 1 | SectionType | `IJSDXSectionType.XSectionType` |
| 2 | ConnectionType | `IJEndCutAttributes.StartConnectionType` |
| 3-5 | PrimaryWebCut (First/Top/Bottom) | `IJSDEndCutData.StartEndCutData` 解析 |
| 6-8 | TopFlangeCut (First/Left/Right) | 同上 |
| 9-11 | BottomFlangeCut (First/Left/Right) | 同上 |
| 12-14 | SecondaryWebCut (First/Top/Bottom) | 同上 |

> 端切数据格式示例: `"3; |2; ; | ; ; | ; ;"`，按 `|` 分割为4组，每组按 `;` 分割为3个子值。

#### 2.7 XPath 匹配逻辑

XML 搜索路径构建为多条件 XPath，每个维度支持精确匹配或通配（`Any`/节点不存在）：

```xml
<!-- XML 规则文件结构示例 -->
<RULE>
  <CUSTOM>
    <Codelists>
      <PrimaryWebCutInfo namespace="UDP">WebCutDrawingTypeCodeList</PrimaryWebCutInfo>
```

## 五、开发要点总结

### Planning 开发要点

1. **模型创建顺序**: StructuralSystem → GridSystem → PlateSystem/StiffenerSystem/MemberSystem → Detail() → 获取 Part
2. **Detail() 必须调用**: PlateSystem 和 StiffenerSystem 创建后必须调用 `Detail()` 才能生成 PlatePart/StiffenerPart
3. **装配体类型**: `New Assembly(parent, type)` 中 type 决定装配体类型（1=Assembly, 2=Block）
4. **服务异步等待**: `PlanningAssignmentService` 和 `CommonPartsService` 是异步服务，需要轮询 `Status` 等待完成
5. **规则获取**: `CatalogBaseHelper.GetRuleDefinition(ruleId, version)` 用于获取焊接规则定义

### ShipDrawings 开发要点

1. **CustomPropertyBase**: 自定义属性必须通过 `[ProgId]` 注册，`GetProperty` 是核心方法
2. **EndCutRule**: XML 规则文件驱动，15维型材信息匹配，支持码表显示名转换
3. **XPath 通配**: 每个匹配维度都支持 `value or not(node) or node='Any'` 三态匹配

### StructuralAnalysis 开发要点

1. **过滤器模式**: AnalysisModel 可通过 Filter 过滤特定对象类型
2. **载荷体系**: LoadCase → LoadCombination → 各种 Load（Force/Moment），支持 Local/Global/ProjectedGlobal 参考系
3. **模型交换**: CIS/STEP 格式支持导出→修改→重新导入的闭环流程
4. **OutOfDateObjects**: 导入前可预览哪些构件截面将发生变化
