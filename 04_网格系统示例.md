## 1. 坐标系统创建模板

网格系统的起点是创建 `CoordinateSystem`（坐标系统）。所有网格轴、平面、圆柱面都挂载在坐标系统下。

```vb
' 获取模型连接
Dim oModel As Model = oPlant.PlantModel
Dim oRootObj As BusinessObject = DirectCast(oModel.RootSystem, BusinessObject)
Dim oModelConn As SP3DConnection = oRootObj.DBConnection

' 创建坐标系统（网格类型）
Dim oCS As New CoordinateSystem(oModelConn, CoordinateSystem.CoordinateSystemType.Grids)

' 设置原点
oCS.Origin = New Position(0, 0, 0)

' 设置名称（通过 INamedItem 接口）
DirectCast(oCS, INamedItem).SetUserDefinedName("MyCS")

' 将坐标系统添加到系统树
Dim oSystem As ISystem = New SystemHelper(oRootObj)
oSystem.AddSystemChild(oCS)
```

> **关键说明**：`CoordinateSystemType` 枚举有两个常用值——`Grids`（通用网格）和 `Ship`（船舶类型）。

## 2. GridAxis 创建与五种轴类型

`GridAxis`（网格轴）是网格平面的载体。通过 `AxisType` 枚举指定轴方向。

### 五种轴类型

| 轴类型 | 枚举值 | 说明 | 关联的平面类型 |
|--------|--------|------|---------------|
| X 轴 | `AxisType.X` | X 方向水平轴 | `GridPlane` |
| Y 轴 | `AxisType.Y` | Y 方向水平轴 | `GridPlane` |
| Z 轴 | `AxisType.Z` | Z 方向垂直轴（高程） | `GridElevationPlane` |
| 径向轴 | `AxisType.Radial` | 径向轴（极坐标） | `GridRadialPlane` |
| 圆柱面轴 | `AxisType.Cylindrical` | 圆柱面轴（极坐标） | `GridCylinder` |

### 创建所有五种轴类型

```vb
Dim oGridAxis As New GridAxis(oCS, AxisType.X)
Dim oGridAxis2 As New GridAxis(oCS, AxisType.Y)
Dim oGridAxis3 As New GridAxis(oCS, AxisType.Z)
Dim oGridAxis4 As New GridAxis(oCS, AxisType.Radial)
Dim oGridAxis5 As New GridAxis(oCS, AxisType.Cylindrical)

oTransactionMgr.Commit("Place Grid Axis")
```

### AddPlane / AddPlanes 批量创建平面

```vb
' 添加单个平面（在坐标值 0 处）
oGridAxis.AddPlane(0)

' 批量添加平面：start=1, end=4, step=1
oGridAxis.AddPlanes(1, 4, 1)   ' 在 1, 2, 3, 4 处创建平面

' 为所有轴批量创建平面
oGridAxis2.AddPlanes(0, 4, 1)       ' Y 轴
oGridAxis3.AddPlanes(0, 4, 1)       ' Z 轴
oGridAxis4.AddPlanes(0, 4, 1)       ' 径向轴
oGridAxis5.AddPlanes(1, 4, 1)       ' 圆柱面轴

oTransactionMgr.Commit("Place Grid Planes")
```

### GridAxis 关键属性

```vb
' 所属坐标系统
Dim oCS2 As CoordinateSystem = oGridAxis.CoordinateSystem

' 单位方向向量
Dim oVector As Vector = oGridAxis.Unitvector

' 轴类型
Dim oAxisType As AxisType = oGridAxis.AxisType
```

## 3. GridPlane 创建与 IPlane / ISurface 接口

`GridPlane` 是最全面的 GridSystem 示例，覆盖了 `IPlane` 接口和 `ISurface` 接口的所有方法。

### GridPlane 创建

```vb
' 在 Y 轴上创建距离为 1 的平面
Dim oGridPlane As New GridPlane(1.0, oGridAxis)
oGridPlane.SetPropertyValue("GridY_1", "IJNamedItem", "Name")

' 批量创建
For i As Integer = 1 To 3
    oGridPlane = New GridPlane(CDbl(i), oGridAxis)
    oGridPlane.SetPropertyValue("GridY_" & i.ToString(), "IJNamedItem", "Name")
Next
```

### IPlane 接口属性

`GridPlane` 实现了 `IPlane` 接口，提供以下几何属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| `RootPoint` | `Position` | 平面的根点 |
| `Normal` | `Vector` | 平面法向量 |
| `UDirection` | `Vector` | U 方向向量 |
| `VDirection` | `Vector` | V 方向向量 |
| `Reversed` | `Boolean` | 是否反转 |
| `BoundaryCount` | `Integer` | 边界数量 |

```vb
' 测试 IPlane 属性
Dim oRootPoint As Position = oGridPlane.RootPoint
Dim oNormal As Vector = oGridPlane.Normal
Dim oUDirection As Vector = oGridPlane.UDirection
Dim oVDirection As Vector = oGridPlane.VDirection
Dim bReversed As Boolean = oGridPlane.Reversed
Dim iBoundaryCount As Integer = oGridPlane.BoundaryCount

' 获取坐标系统
Dim oPlaneCs As CoordinateSystem = oGridPlane.Axis.CoordinateSystem
```

### ISurface 接口方法

`GridPlane` 同时实现了 `ISurface` 接口，提供曲面计算方法：

| 方法 | 说明 |
|------|------|
| `ScopeNormal(scope, normal)` | 获取范围法向量 |
| `IsPointOn(position)` | 判断点是否在面上 |
| `Intersect(curve, results, code)` | 面与曲线相交计算 |
| `Intersect(surface, curves, code)` | 面与面相交计算 |
| `DistanceBetween(surface, dist, srcPos, inPos)` | 面到面的距离 |
| `DistanceBetween(curve, dist, srcPos, inPos)` | 面到曲线的距离 |
| `DistanceBetween(point, dist, inPos)` | 面到点的距离 |

```vb
' ScopeNormal — 范围法向量
Dim vNormal As Vector = Nothing
Dim eScope1 As SurfaceScopeType
oGridPlane.ScopeNormal(eScope1, vNormal)

' IsPointOn — 点是否在面上
Dim oPointOn As New Position(0, 0.2, 0)
Dim bIsPointon As Boolean = oGridPlane.IsPointOn(oPointOn)

' Intersect — 面与曲线相交
Dim eIntersectCode As GeometryIntersectionType
Dim oCurveBOColl As New Collection(Of BusinessObject)()
oXPlane.Intersect(DirectCast(oGridLine, ICurve), oCurveBOColl, eIntersectCode)

' Intersect — 面与面相交
Dim oCurveColl As New Collection(Of ICurve)
oGridPlane.Intersect(DirectCast(oXPlane, ISurface), oCurveColl, eIntersectCode)

' DistanceBetween — 面到面
Dim dDist As Double
Dim oPosSrcPos As Position = Nothing
Dim oPosInPos As Position = Nothing
oGridPlane.DistanceBetween(DirectCast(oTGridPlane, ISurface), dDist, oPosSrcPos, oPosInPos)

' DistanceBetween — 面到曲线
Dim dDist1 As Double
oTGridPlane.DistanceBetween(DirectCast(oGridLine, ICurve), dDist1, oPosSrcPos1, oPosInPos1)

' DistanceBetween — 面到点
Dim dDist2 As Double
Dim oPoint3d As New Point3d(2, 2, 6)
oGridPlane.DistanceBetween(oPoint3d, dDist2, oPosInPos2)
```

## 4. 交叉元素生成

网格系统的核心特性是通过平面与平面的**交叉**自动生成线（GridLine）和弧（GridArc）。

### GridPlane.CreateGridLine — 平面交叉生成线

`GridPlane`（X/Y方向平面）与 `GridElevationPlane`（Z方向高程面）交叉生成 `GridLine`：

```vb
' Y 方向平面与高程面交叉 → GridLine
Dim oGridLine As GridLine = oGridPlane.CreateGridLine(oElevation)

' 获取已存在的交叉线
oGridLine = oGridPlane.GetGridLine(oElevation)

' 获取平面上的所有交叉线
Dim oGridLines As List(Of GridLine) = oGridPlane.GridLines
```

### GridElevationPlane 的交叉操作

`GridElevationPlane`（高程面）可以与 X/Y 平面交叉生成 GridLine，也可以与圆柱面交叉生成 GridArc：

```vb
' 高程面与 Y 平面交叉 → GridLine
oGridLine = oElevation.CreateGridLine(oGridYPlane)

' 高程面与 X 平面交叉 → GridLine
oGridLine = oElevation.CreateGridLine(oGridXPlane)

' 获取已存在的交叉线
Dim oGetGridLine As GridLine = oElevation.GetGridLine(DirectCast(oGridYPlanes(0), GridPlaneBase))

' 高程面与圆柱面交叉 → GridArc
Dim oGridArcs As List(Of GridArc) = oElevation.CreateGridArcs(oCPlane)

' 获取已存在的交叉弧
oGridArcs = oElevation.GetGridArcsOnCylinder(oCPlane)

' 获取所有交叉线和弧
Dim oAllGridLines As List(Of GridLine) = oElevation.GridLines
Dim oAllGridArcs As List(Of GridArc) = oElevation.GridArcs
```

### GridCylinder.CreateGridArcs — 圆柱面交叉生成弧

`GridCylinder`（圆柱面）与高程面交叉生成 `GridArc`：

```vb
' 圆柱面与高程面交叉 → GridArc
Dim oGridArcs As List(Of GridArc) = oGridCylinder.CreateGridArcs(oElevation)

' 获取已存在的交叉弧
oGridArcs = oGridCylinder.GetGridArcsOnElvationPlane(oElevation)

' 获取所有弧
oGridArcs = oGridCylinder.GridArcs
```

## 5. GridsAPIHelper 查询

`GridsAPIHelper` 提供从数据库查询坐标系统的静态方法，有三种重载：

```vb
Dim oGridHelper As GridsAPIHelper = New GridsAPIHelper()

' 1. 查询网格类型坐标系统
Dim oCSCollection As Collection(Of CoordinateSystem)
oCSCollection = oGridHelper.GetCoordinateSystemsFromDB(
    oModelConn, CoordinateSystem.CoordinateSystemType.Grids)

' 2. 查询船舶类型坐标系统
oCSCollection = oGridHelper.GetCoordinateSystemsFromDB(
    oModelConn, CoordinateSystem.CoordinateSystemType.Ship)

' 3. 查询所有坐标系统（不指定类型）
oCSCollection = oGridHelper.GetCoordinateSystemsFromDB(oModelConn)
```

> **使用场景**：在需要获取已存在的坐标系统进行修改或查询时使用，而不是创建新的。

## 6. 网格系统层次结构图

```
┌─────────────────────────────────────────────────────────────────────┐
│                     网格系统完整层次结构                                │
└─────────────────────────────────────────────────────────────────────┘

  CoordinateSystem (坐标系统)
  │   · Origin (原点)
  │   · CoordinateSystemType (Grids / Ship)
  │   · INamedItem.SetUserDefinedName()
  │
  ├── GridAxis (网格轴) × N
  │   │   · AxisType: X / Y / Z / Radial / Cylindrical
  │   │   · Unitvector (单位向量)
  │   │   · AddPlane(value) / AddPlanes(start, end, step)
  │   │
  │   ├── AxisType.X → GridPlane (X方向网格平面)
  │   │                  · 实现 IPlane + ISurface 接口
  │   │                  · CreateGridLine(elevation) → GridLine
  │   │
  │   ├── AxisType.Y → GridPlane (Y方向网格平面)
  │   │                  · 实现 IPlane + ISurface 接口
  │   │                  · CreateGridLine(elevation) → GridLine
  │   │
  │   ├── AxisType.Z → GridElevationPlane (高程平面)
  │   │                  · CreateGridLine(gridPlane) → GridLine
  │   │                  · CreateGridArcs(cylinder) → GridArc
  │   │
  │   ├── AxisType.Radial → GridRadialPlane (径向平面)
  │   │                       · CreateGridLine() → GridLine
  │   │
  │   └── AxisType.Cylindrical → GridCylinder (网格圆柱面)
  │                               · 实现 ISurface 接口
  │                               · CreateGridArcs(elevation) → GridArc
  │
  ├── 交叉元素（自动生成）
  │   │
  │   ├── GridLine (网格线)
  │   │   │   由 GridPlane ∩ GridElevationPlane 生成
  │   │   │   或由 GridRadialPlane ∩ GridElevationPlane 生成
  │   │   │
  │   │   ├── DefineBy2Points(start, end)
  │   │   ├── DefineByStartDir(start, direction)
  │   │   ├── DefineByStartDirLength(start, direction, length)
  │   │   ├── StartPoint / EndPoint / Direction / Length
  │   │   └── Infinite (是否无限延伸)
  │   │
  │   └── GridArc (网格弧)
  │       │   由 GridCylinder ∩ GridElevationPlane 生成
  │       │
  │       ├── Origin (原点)
  │       ├── Quadrant (象限)
  │       └── RadialCylinder (所属圆柱面)
  │
  └── GridsAPIHelper (查询辅助)
      └── GetCoordinateSystemsFromDB(connection, [type])
```

## 7. GridLine 几何定义

`GridLine` 是由平面交叉自动生成的线元素，但也支持手动重新定义其几何形状。

### 三种几何定义方式

```vb
' 方式一：两点定义
Dim oStart As New Position(1, 1, 1)
Dim oEnd As New Position(2, 2, 2)
oGridLine.DefineBy2Points(oStart, oEnd)

' 方式二：起点 + 方向（无限线）
Dim oDirection As Vector = oGridLine.Direction
oGridLine.DefineByStartDir(oPos, oDirection)

' 方式三：起点 + 方向 + 长度（有限线）
Dim Length As Double = 1.0
oGridLine.DefineByStartDirLength(oPos, oDirection, Length)

oTransactionMgr.Commit("Place Grid Line")
```

### 关键属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `StartPoint` | `Position` | 起点（仅当 Infinite = False 时有效） |
| `EndPoint` | `Position` | 终点（仅当 Infinite = False 时有效） |
| `Direction` | `Vector` | 方向向量 |
| `Length` | `Double` | 长度 |
| `Infinite` | `Boolean` | 是否无限延伸 |
| `RootPoint` | `Position` | 根点 |
| `Centroid` | `Position` | 质心 |
| `Form` | `CurveFormType` | 曲线形式 |
| `Scope` | `CurveScopeType` | 范围 |
| `ElevationPlane` | `GridElevationPlane` | 所属高程平面 |
| `IntersectingPlane` | `GridPlaneBase` | 相交平面 |

```vb
' 判断是否为有限线
If oGridLine.Infinite <> True Then
    Dim oStartPoint As Position = oGridLine.StartPoint
    Dim oEndPoint As Position = oGridLine.EndPoint
End If

' 获取几何属性
Dim oRootpoint As Position = oGridLine.RootPoint
Dim oDirection As Vector = oGridLine.Direction
Dim oCentroid As Position = oGridLine.Centroid
Dim oForm As CurveFormType = oGridLine.Form
Dim dLen As Double = oGridLine.Length
Dim oScope As CurveScopeType = oGridLine.Scope

' 获取关联平面
Dim oElevPlane As GridElevationPlane = oGridLine.ElevationPlane
Dim oIntersectingPlane As GridPlaneBase = oGridLine.IntersectingPlane
```

### 边界操作

```vb
Dim oBound1 As BusinessObject = Nothing
Dim oBound2 As BusinessObject = Nothing
oGridLine.GetBoundaries(oBound1, oBound2)
```

## 通用属性（所有网格元素共有）

所有网格元素（GridAxis、GridPlane、GridElevationPlane、GridCylinder、GridRadialPlane、GridLine、GridArc）均继承自公共基类，共享以下属性：

| 属性 | 说明 |
|------|------|
| `Range` | 包围盒 |
| `Matrix` | 变换矩阵 |
| `Origin` | 原点位置 |
| `SystemChildren` | 子元素集合 |
| `SystemParent` | 父系统 |
| `NestingLevel` | 嵌套级别（Primary / Secondary） |
| `DistanceFromOrigin` | 距原点距离 |
| `Type` | 元素类型 |
| `EDIFlag` | EDI 标志 |
| `OriginAppName` | 来源应用名称 |
| `OriginOid` | 来源对象 ID |
| `OriginAppDesc` | 来源应用描述 |
| `OriginAppId` | 来源应用 ID |

### NestingLevel 属性

```vb
Dim oNestingLevel As NestingLevelType = oGridPlane.NestingLevel
oGridPlane.NestingLevel = NestingLevelType.Secondary
oTransactionMgr.Commit("Modify Grid Plane")
```

### GetReference 方法

各平面类型提供 `GetReference***` 方法获取参考平面：

```vb
' GridPlane
Dim oGridRefPlane As GridPlane = oGridPlane.GetReferenceGridPlane(
    ReferenceType.Origin, NestingLevelType.Secondary)

' GridElevationPlane
Dim oGridElevRef As GridElevationPlane = oElevation.GetReferenceElevationPlane(
    ReferenceType.Origin, NestingLevelType.Primary)

' GridCylinder
Dim oGridRefCylinder As GridCylinder = oGridCylinder.GetReferenceGridCylinder(
    ReferenceType.Origin, NestingLevelType.Primary)
```
